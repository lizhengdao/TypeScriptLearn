其实TS用的很多东西跟ES6是高度重合的，包括箭头函数/解构赋值/模块/类的部分特性
不重合的是类型声明 接口的灵活实现 还有一些补充的 关键字/抽象类/函数重载/泛型

写了一些语法`demo`,供大家研究
